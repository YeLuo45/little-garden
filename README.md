# 小花园 - 儿童成长陪伴游戏

一款面向 3-12 岁儿童的养成类网页游戏。玩家可以照顾宠物、种植花草，体验温馨的花园生活。

**GitHub Pages 访问地址**: https://yeluo45.github.io/little-garden/

---

## 游戏特性

### 宠物系统
- 可爱的小狗「小白」和小猫「小橘」
- 喂食、抚摸、玩耍多种互动
- 宠物心情和饥饿值实时显示

### 种植系统
- 向日葵、玫瑰、郁金香等多种作物
- 种子 → 发芽 → 开花 → 成熟的完整生长周期
- 点击空地播种，收获后获得金币

### 双模式设计
- **幼儿模式（3-6岁）**：操作简单，图标为主
- **儿童模式（7-12岁）**：任务引导，组合操作

## 本地运行

### 方式一：直接打开
双击 `index.html` 用浏览器打开即可游戏。

### 方式二：本地服务器
```bash
# Python 3
python3 -m http.server 8080

# 然后访问 http://localhost:8080
```

## 技术栈

- **引擎**: 纯 HTML5 Canvas + Vanilla JavaScript
- **渲染**: Canvas 2D（程序化绘制角色和场景）
- **音效**: Web Audio API（程序化合成）
- **存储**: localStorage（自动保存）
- **部署**: GitHub Pages（无需构建）

## 游戏操作

| 操作 | 说明 |
|------|------|
| 点击宠物 | 增加好感 |
| 点击喂食 | 消耗宝石喂食 |
| 点击抚摸 | 增加宠物心情 |
| 点击玩耍 | 增加宠物心情 |
| 点击空地 | 播种作物 |
| 点击作物 | 浇水加速生长 |
| 点击成熟作物 | 收获获得金币 |

---

## 开发说明

游戏使用纯 JavaScript 编写，无需任何构建工具或外部依赖。

**画布尺寸**: 720 × 1280（竖屏）
**目标帧率**: 60 FPS

如需修改游戏内容，可直接编辑 `game.js` 文件。
